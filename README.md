# 在线考试管理系统 python780

## 项目概述

在线考试管理系统是一款基于Python3和Django框架开发的Web应用，结合Vue.js前端框架，实现了学院、班级、考试科目、学生、教师信息的全面管理。

## 技术栈

- 前端：Vue.js
- 后端：Python3、Django
- 数据库：MySQL 8.0
- 开发工具：PyCharm、Navicat

## 功能模块

- **学院信息管理**：实现对学院基础信息的增删改查。
- **班级信息管理**：管理班级信息，便于进行统一的教学管理。
- **考试科目管理**：设置和管理考试科目，支持科目信息的维护。
- **学生信息管理**：对学生信息进行管理，包括考试成绩的记录。
- **教师信息管理**：维护教师基本信息，并与课程管理相结合。
- **校园考试管理**：全面管理校园内的考试活动。

## 系统角色

系统主要分为以下角色：
- 学生
- 教师
- 管理员

各角色拥有不同权限，共同维护考试管理系统的正常运行。

## 运行环境

- Python3.0以上
- MySQL 8.0
- Node.js（用于支持Vue.js）

## 部署步骤

1. 安装Python3及其依赖环境。
2. 安装MySQL 8.0，并导入初始数据库。
3. 安装Node.js及Vue.js相关环境。
4. 部署后端应用，根据文档配置相关参数。
5. 部署前端应用，确保与后端服务通信无误。
